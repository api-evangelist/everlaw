# Everlaw

Everlaw is a cloud-based litigation and eDiscovery platform offering a REST API for managing cases, uploading evidence, running predictive coding, and collaborating on document review.

## API

The Everlaw REST API provides programmatic access to the eDiscovery platform, enabling automation of case management, document uploads, billing reporting, and analytics. API keys are generated by Organization Admins.

- **API Documentation (US):** https://api.everlaw.com/docs/
- **API Documentation (UK):** https://api.everlaw.co.uk/docs/
- **Support / Knowledge Base:** https://support.everlaw.com/hc/en-us
- **MCP Server:** https://support.everlaw.com/hc/en-us/articles/49986656959771-Everlaw-MCP-Server
- **Status:** https://status.everlaw.com
- **Blog:** https://www.everlaw.com/blog/
- **Website:** https://www.everlaw.com

## Authentication

Authentication uses API keys. Keys are generated on the Organization Admin page and are scoped to the organization. A valid organization API key grants access to all API endpoints.

## Rate Limits

- 25 requests per second per authenticating user account
- Requests exceeding this limit receive HTTP 429 Too Many Requests
- A stricter rate limit applies to failed authentication attempts
- Recommended retry strategy: exponential backoff

## Pricing

Everlaw uses a monthly per-gigabyte pricing model with no per-user, processing, or module fees. All features — including AI tools — are included at no additional charge. API access is included with a subscription.

- Estimated $18-35/GB per month (third-party estimates; actual pricing requires a quote)
- Volume discounts at 500GB+
- No setup, processing, user, training, exporting, or production fees

Contact Everlaw for pricing: https://www.everlaw.com/request-demo/
